## 🎲 Aegis Faker v1.0.0 Production Release

**Aegis Faker**, a premium, high-performance hardware identity masking module built using Jetpack Compose and Material 3 design systems. 

### 🚀 Key Features
* **1-Click Identity Randomization:** instant generation of unique Android IDs and Serial numbers.
* **Modern Dashboard:** High-end cyber-utility aesthetic interface.
* **Core Hooking Engine:** Secure interception of hardware parameters via LSPosed.
* **System Safeguards:** Core OS processes are explicitly isolated to ensure complete system stability and eliminate risk of bootloops.

### 🛠️ Installation Notes
1. Ensure your device is rooted with Magisk or KernelSU.
2. Download and install the **`AegisFaker_v1.0.apk`** attached below.
3. Open the **LSPosed Manager App**, toggle the module **On**, and check your target applications.
4. Reboot your device or restart the targeted apps to apply changes.
